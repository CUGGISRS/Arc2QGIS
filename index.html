Welcome,
This guide-in-the-making is meant for ArcGIS users wanting to make the switch to open source solutions.
If you know how to do operations in ArcGIS and want to know how to them in QGIS, GDAL, GEOS, GRASS or any other way, you're welcome to see if the solution is already here.
If you want to contribute, you can edit you HTML content here, or upload a word document which i'll convert to HTML so everyone can access it. Any and all help is welcome, as are requests and questions in the issues section of the repository in github.
